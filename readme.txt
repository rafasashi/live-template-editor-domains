LTPLE template editor  

== Description ==

Live Template Editor Domains allows you to edit template profils.

== Installation ==

Installing "Live Template Editor Domains" can be done either by searching for "Live Template Editor Domains" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
